Reset HealBot


To set absolute defaults for Healbot on all Accounts and Realms, do the following in order:

1: Completely Logout of WoW
2: Download Reset_HealBot.zip

On Windows
3: Unzip Reset_HealBot.zip in the root WoW folder (ie. C:\Program Files\World of Warcraft)
4: Run Reset_HealBot.bat
5: Type “D” to delete Healbot’s WTF files
6: That’s it, logon to WoW and set Healbot Spells, Buffs, Cures and Skins


On Mac
3: Unzip Reset_HealBot.zip in the root WoW folder (ie. /Applications/World of Warcraft)
4: Run Reset_HealBot.pl using the Terminal application located in the Applications/Utilities Folder, and type: perl /Applications/'World of Warcraft'/Reset_HealBot.pl
5: Type “Y” to delete Healbot’s WTF files
6: That’s it, logon to WoW and set Healbot Spells, Buffs, Cures and Skins
